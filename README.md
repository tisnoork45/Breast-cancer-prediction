# Breast Cancer Prediction using Decision Tree

## Overview
This machine learning project predicts whether a breast tumor is benign or malignant using the Breast Cancer Wisconsin dataset.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
- Decision Tree Classifier (CART)

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Performance Evaluation

## Results
The model successfully classifies breast cancer cases as benign or malignant based on diagnostic features.
